## 🐛 常见问题排查

### 问题1：翻译不显示
**可能原因**：
- 忘记导入i18n配置
- 键名错误
- useTranslation hook未正确使用

**解决方法**：
```typescript
// 确保在_layout.tsx导入
import '../src/i18n';

// 在组件中正确使用
const { t } = useTranslation();
```

### 问题2：语言切换无效
**可能原因**：
- AsyncStorage权限问题
- i18n缓存问题

**解决方法**：
- 检查AsyncStorage是否可用
- 清除应用数据重新测试

### 问题3：系统语言检测失败
**可能原因**：
- expo-localization配置问题

**解决方法**：
```typescript
// 检查设备语言
console.log('Device locale:', Localization.locale);
```
