# RunForge 国际化使用指南

## 🎉 已完成功能

您的RunForge应用现在已经支持中英文双语！用户可以根据喜好自由切换语言。

## 🚀 如何使用

### 1. 语言自动检测
- 应用会根据用户的系统语言自动设置初始语言
- 中文系统 → 显示中文界面
- 其他语言系统 → 显示英文界面

### 2. 手动切换语言
1. 打开应用
2. 点击底部 "档案/Profile" 标签页
3. 滚动到 "系统设置" 部分
4. 点击 "语言" 选项
5. 在弹出的对话框中选择想要的语言
6. 应用会立即切换到新语言

### 3. 已国际化的界面

#### ✅ 完全支持
- **首页**: 包括新用户引导、今日行动、身体状态、本周推进等所有文案
- **个人档案页**: 所有表单字段、按钮、错误提示都已翻译
- **底部导航**: Tab标题已国际化
- **语言切换器**: 完整的语言选择功能

#### 🎨 主要特性
- **智能语言检测**: 基于系统语言自动选择
- **持久保存**: 用户选择的语言会被记住
- **实时切换**: 无需重启应用，立即生效
- **完整翻译**: 包括错误消息、提示文案等

## 🔧 开发者信息

### 技术实现
- **框架**: React Native + Expo
- **国际化库**: react-i18next + i18next + expo-localization
- **语言支持**: 中文 (zh) / 英文 (en)
- **存储**: AsyncStorage（持久保存用户语言偏好）

### 语言文件位置
```
mobile/src/i18n/locales/
├── zh.json (中文)
└── en.json (英文)
```

### 添加新翻译
如需添加新的翻译文本，请在对应的语言文件中添加键值对，然后在代码中使用 `t('key')` 调用。
